# 💭 Why did I build peerY?

When I started learning web development...

I was completely lost.

Every day looked something like this:

```
YouTube
      ↓
Udemy
      ↓
FreeCodeCamp
      ↓
Documentation
      ↓
Discord
      ↓
Stack Overflow
      ↓
Reddit
      ↓
ChatGPT
      ↓
Repeat.
```

I wasn't learning.

I was just **collecting tutorials**.

Some days I'd watch React.

The next day Node.js.

Then Docker.

Then TypeScript.

Then AI.

Then System Design.

At the end of the week...

I knew a little bit of everything...

and nothing deeply enough to build something meaningful.

The biggest problem wasn't the resources.

There are thousands of amazing resources online.

The problem was...

I had **no direction.**

---

Then came another problem.

I wanted to build projects.

But...

I had no teammates.

No developer friends.

No mentor.

No community.

No one to review my code.

No one to say,

> "You're going in the wrong direction."

So I built everything alone.

Sometimes I learned.

Sometimes I gave up.

Sometimes I restarted from scratch.

And I realized...

Thousands of developers are going through the exact same journey.

---

# 💡 So I asked myself...

What if one platform could help developers...

✅ Learn

✅ Find teammates

✅ Discover projects

✅ Get AI guidance

✅ Build together

✅ Grow together

Instead of switching between ten different tools...

What if everything lived in one place?

That's how **peerY** was born.

# 🚀 Getting Started

Clone the repository

```bash

```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
